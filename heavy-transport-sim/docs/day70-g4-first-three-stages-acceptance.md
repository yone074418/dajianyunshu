## 6. G4 验收主流程

```
登录或进入学生测试模式
→ 查看运输任务介绍 ✅
→ 查看货物参数和360°/尺寸信息 ✅
→ 完成必要知识学习或确认学习状态 ✅
→ 查看当前步骤提示 ✅
→ 进入简单配车 ✅
→ 选择车辆组合方式 ✅
→ 查看组合动画或演示 ✅
→ 选择牵引车 ✅
→ 选择挂车轴线数和纵列数 ✅
→ 执行简单配车规则检查 ✅
→ 正确方案允许继续，错误方案必须修正或显示明确原因 ✅
→ 进入路线勘测 ✅
→ 查看三条路线和障碍列表 ✅
→ 完成或复查五类障碍测量 ✅
→ 查看高度、圆弧弯道规则结论 ✅
→ 查看直交弯道、坡道牵引力、桥梁承载规则结论 ✅
→ 查看三条路线建议 ✅
→ 切换三条路线并重复查看结论 ✅
→ G4结论：任务、配车、勘测基本连续完成
```
