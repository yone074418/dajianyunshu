# Day30 自制低模占位资产

本目录仅包含第30天任务范围内的自制简单几何占位低模，覆盖：

- AR-001 大件货物模型
- AR-002 6x6 与 8x8 牵引车
- AR-003 全挂车、半挂车、自行式模块运输车
- AR-004 模块挂车与拼接部件

这些文件是项目自制占位资产，不代表正式美术资产，不代表已经完成商业授权采购，也不代表最终 GLB 交付物。当前文件使用 Wavefront OBJ 文本格式，目的是在不安装新依赖、不下载外部素材、不执行 Day34 GLB 导出与贴图压缩的前提下，提供可被 Blender 打开的低模占位文件。

正式文件格式目标仍按 Day29 清单执行为 GLB。后续若需要导出 GLB、统一坐标/比例/原点、压缩贴图或接入应用，应在对应计划日执行。
