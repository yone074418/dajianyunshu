# G1资产需求基线

> 基线编号：ASSET-REQ-001 / c0.3.1
>
> 批准依据：PCR-005、PDEC-005
> 性质：G1需求输入，不代表资产已取得、已授权或已制作

## 1. 边界

G1只冻结资产类别、教学用途、使用阶段、最低可辨识标准和需求证据。来源、许可、目标文件大小、面数/贴图预算、制作方式、负责人和完成状态由第29天资产执行清单管理；第29天不得静默增删本基线类别，变化须走CR。

## 3. 第29天执行清单必填项

每个AR条目在第29天拆分为具体文件或制作任务，并填写：来源、许可、制作/采购方式、目标格式、目标大小、面数/贴图预算、负责人、状态和验收证据。未知项必须显式标记，不能写成“已完成”。

## 4. 验收

- AR-001—019均能追溯到论文、实施计划或前11天候选文档。
- 未新增语音、即时通信、多人协同、移动端或其他OUT/BAN/FUT资产。
- 未声明任何版权、许可、文件预算或制作完成状态。
- CON-004仅在计划口径同步且本清单纳入候选基线后关闭。
