## 案例背景

某电力工程需要将一台大型电力变压器从制造厂运输至变电站施工现场。变压器属于不可拆解的大件货物，外形尺寸和重量均超出普通公路运输标准，需要制定专门的大件运输方案。学生需要根据货物参数、路线约束和车辆条件，完成从任务分析到运输交付的全过程仿真实验。

## 运输目标

1. 根据货物尺寸和重量选择合适的运输车辆
2. 分析起点到终点的可选运输路线
3. 检查路线上的高度、弯道、坡度、桥梁承载等约束
4. 确定最终车组和运输方案
5. 完成货物装车和绑扎加固
6. 执行运输仿真并验证方案可行性

## 教学目标

- 理解大件运输方案制定的基本流程
- 掌握货物参数对运输方案的影响
- 了解路线约束检查的关键要素
- 培养大件运输安全意识

## 案例约束

| 约束类型 | 说明 |
|----------|------|
| 高度约束 | 运输路线上的限高架、桥梁净空高度必须大于货物高度加车辆高度加安全余量 |
| 弯道约束 | 弯道半径必须满足车辆转弯要求，特别注意直角弯和环岛路段 |
| 坡度约束 | 坡度不得超过车辆爬坡能力限制，下坡需考虑制动安全 |
| 桥梁承载 | 沿线桥梁承载能力必须大于车货总重，需逐桥核算 |
| 轴载约束 | 各轴轴载不得超过道路承载标准，需合理分布货物重心 |
| 绑扎约束 | 绑扎方案必须满足大件运输固定要求，防止运输途中位移和倾覆 |

## 教学简化声明

- 本案例为教学简化案例，使用虚构地名，不代表真实工程路线
- 货物参数基于教学需要设定，实际工程应以设计图纸为准
- 路线约束仅为教学演示，实际大件运输需依据现场勘测数据
- 学生应重点关注大件运输方案的整体合理性，而非精确数值计算
